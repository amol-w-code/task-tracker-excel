## ✨ Key Features

1. **🔐 Multi-User Authentication & Email Verification**:
   - Secure signup with **6-Digit Email Verification Code** security.
   - Every registered user gets their own private, isolated Habit Matrix and personal streak counters.
2. **📱 Reference Floating Curved Navigation Bar**:
   - Fixed curved navigation pill bar with an **Elevated Center Action Button (+)** inside a glowing circle notch.
   - Touch-friendly `42x42px` checkboxes optimized for smartphones and iPad/Android tablets.
3. **📥 Progressive Web App (PWA) Mobile App Installation**:
   - Visit the live app on any mobile phone or tablet and tap **"📱 Install Mobile App"** to add TaskPulse directly to your home screen as a standalone fullscreen application!
4. **📗 Exact Excel (.xlsx) Report Export**:
   - 1-click download generates a customized `.xlsx` spreadsheet matching the exact Habit Matrix checkmark grid (`☑` / `☐`).

## 🚀 Running Locally

```bash
# 1. Clone repository
git clone https://github.com/amol-w-code/task-tracker-excel.git
cd task-tracker-excel

# 2. Install packages
npm install

# 3. Start server
npm start
```
Open **http://localhost:3000** in your browser!
